# Portfolio Selection Notes

## Included

- Java GUI project, rewritten and anonymized.
- WSN topology/MST simulation, refactored into a reusable Python script.
- Engineering coursework summaries rewritten as portfolio notes.

## Excluded From Public Repo

- Raw `.doc`, `.docx`, `.ppt`, `.pptx`, `.pdf`, `.mp4` coursework files.
- Files with student ID or classmates' names.
- `__MACOSX`, `.DS_Store`, IDE metadata, compiled `.class` files.
- Personal or private documents.

## Why

Public GitHub should communicate engineering ability, not look like a raw homework archive. This version is safer and more professional for resume use.
